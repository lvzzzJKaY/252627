# 10-Minute Presenter Note

## Suggested flow

1. **0:00-1:00** - Open the landing section and read the intro sentence out loud.
2. **1:00-7:00** - Click each chapter card (any order):
   - Read "What Happens" quickly first.
   - Then read "Why It Matters" + "Key Theme".
   - Ask the short discussion question and let classmates answer for 20-30 seconds.
   - Use the chapter interaction button as a quick vote/check-in.
3. **7:00-9:00** - Use the final reflection poll and ask for 2-3 student responses.
4. **9:00-10:00** - Summarize: **myth -> waiting -> identity** as Tara's turning arc.

## Presentation tip

If time is short, prioritize the chapter subtitles and "Why It Matters" boxes. Those carry the core interpretation clearly.
